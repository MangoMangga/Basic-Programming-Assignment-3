# Airline Ticket Reservation System
## Description

## Github Link: https://mangomangga.github.io/Basic-Programming-Assignment-3/

This program is a simple Airline Ticket Reservation System written in Python.
It allows users to book seats, view seat availability, search passengers, cancel bookings, and save booking data.

How It Works

- Seats are stored using a 5Ã—6 2D array (O = available, X = booked).
- Passenger details are managed using a Passenger class.
- Users interact with the system through a menu-based interface.
- Booking data is saved to and loaded from a text file (booking_ticket.txt).

Main Features

- Book airline tickets
- View seat layout
- Search passenger by name
- Cancel booking
- Save and load booking records
